# 🚀 AlgoViz Pro — Game of Algos

**AlgoViz Pro** is an advanced, interactive web-based visualization platform designed to make learning algorithms intuitive, engaging, and practical. Built using **HTML5, CSS3, and JavaScript**, the platform transforms complex algorithmic concepts into dynamic, step-by-step visual experiences.

---

## 📌 Overview

Understanding algorithms purely through theory can be challenging. AlgoViz Pro bridges this gap by combining **visual animations, code implementations, and conceptual explanations** in one unified platform.

It is designed for:

* Students studying **Design and Analysis of Algorithms (DAA)**
* Developers revising core concepts
* Anyone who prefers **visual learning over static theory**

---

## 🎯 Key Features

* 🎬 **Interactive Visualizations**
  Real-time animations that demonstrate how algorithms work internally.

* ⚡ **35+ Algorithms Covered**
  Organized into 7 major categories:

  * Sorting
  * Searching
  * Graph Algorithms
  * Greedy Methods
  * Dynamic Programming
  * Divide and Conquer
  * Data Structures

* 🧠 **Multi-Layer Learning Approach**
  Each algorithm includes:

  * Animated visualization
  * Pseudocode
  * JavaScript implementation
  * Step-by-step explanation
  * Time & space complexity analysis

* 🎛️ **Speed Control Engine**
  Adjust animation speed to learn at your own pace.

* 📊 **Dedicated Visualization Modules**

  * Bar visualizer for sorting algorithms
  * Graph canvas for:

    * BFS & DFS
    * Dijkstra’s Algorithm
    * Kruskal’s & Prim’s Algorithms
  * Dynamic Programming table renderer:

    * 0/1 Knapsack
    * Longest Common Subsequence (LCS)
  * Huffman Coding Tree builder
  * Linked List & Tree (BST/AVL) interactive panels
  * Strassen’s Matrix Multiplication visualizer

* 🧭 **User Interface**

  * Animated splash screen
  * Sidebar navigation with categorized algorithms
  * Clean and responsive design

## 🚀 Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/AlgoViz-Pro.git
   ```

2. Navigate to the project folder:

   ```bash
   cd AlgoViz-Pro
   ```

3. Open `index.html` in your browser:

   ```bash
   start index.html
   ```

---

## 📖 How to Use

1. Launch the application in your browser
2. Select a category from the sidebar
3. Choose an algorithm
4. Interact with:

   * Visualization canvas
   * Code panel
   * Step-by-step explanation
5. Adjust animation speed for better understanding
